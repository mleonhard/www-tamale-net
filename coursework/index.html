Division via Lookup-Table, Parity Generator

This assignment proved to be very challenging due to bugs in MythSim and ambiguities in the assignment description. To work around the MythSim bugs, I designed a MythSim Assembly Language and wrote an assembler in Python. This allowed me to complete Part 2 of the assigment in a short amount of time.

  • Part 1: Micro-code implementing a division instruction using a lookup table in memory
  • Part 2: A complex machine language program to calculate odd-parity using the division instruction
  • MythSim visual microcode simulator
  • Assignment Description: proj5.assignment.pdf and proj5.assignment.note.txt
  • My Submission:
    • assemble.py - Myth Assembler!
    • proj5.asm - Assembly source code for Part 2
    • proj5.mem
    • proj5.src.ucode - Microcode for Part 1, including comments
    • stripcomments.py - Strips comments out of microcode files to avoid MythSim's parsing errors
    • proj5.ucode
